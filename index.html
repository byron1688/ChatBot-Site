1 System Architecture

Multi-Agent Routing & RAG on 32 GB

The core challenge: run multiple LLMs simultaneously for different tasks (routing, generation, memory extraction, TTS) within 32 GB of unified memory. Every model choice is a tradeoff between quality, latency, and memory footprint.

The Routing Pipeline

Each user message passes through a multi-layer classification cascade before hitting the generation model. The goal: route fast, cheap messages to tiny models and reserve the 8B for complex tasks.

2 3D Avatar & Animation

LLM-Driven Interactive VRM Avatar

Having taken a Video Game Design and Computer Graphics course where I worked with Unity and animation pipelines, I wanted to bring the chat companion to life with a real-time 3D avatar whose behavior responds to conversation content.

3 Real-Time TTS Pipeline

Multi-Engine Voice Synthesis on Constrained Hardware

The TTS system supports 5 engines (including bilingual Orpheus variants) with automatic language-based routing and per-skill voice customization. A significant engineering effort went into making streaming audio work smoothly on M1 Max.

4 Voice Input & Speech-to-Text

Local Whisper-Based STT

Voice input uses faster-whisper running locally on CPU with INT8 quantization for bilingual Chinese/English transcription.

5 Image Generation

Stable Diffusion via Draw Things

Image generation runs through Draw Things on macOS, providing an SD WebUI-compatible HTTP API for local Stable Diffusion inference.
